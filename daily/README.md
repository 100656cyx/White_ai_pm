# Daily Learning Logs

这里记录每天的 AI 学习、实操和思考。

建议每天只花 5-10 分钟填写，不追求完整文章，重点是留下可被复盘和自动汇总的素材。

## How to Use

1. 每天复制 `daily/template.md`。
2. 文件命名为 `YYYY-MM-DD.md`，例如 `2026-07-03.md`。
3. 填写当天学习、实操、问题和下一步。
4. 每晚自动任务会读取当天记录，整理到 `summaries/`。

## Suggested Daily Standard

每天至少记录：

- 今天学了什么
- 今天动手做了什么
- 遇到什么问题
- 对 AI 产品经理岗位有什么新理解
- 明天准备推进什么
